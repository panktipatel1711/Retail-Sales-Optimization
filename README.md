## 🚀 Key Technical Features
- **Predictive Modeling:** Time-series forecasting using Random Forest.
- **Supply Chain Optimization:** Automatic Safety Stock and Reorder Point (ROP) calculation.
- **Modern UX:** Sleek Dark-Theme dashboard built with Streamlit and Plotly.
- **Data Analysis:** Dedicated EDA notebook for trend and seasonality discovery.

---

## 🛠️ Installation & Setup
1. **Clone Repo:** `git clone https://github.com/yourusername/retail-inventory-optimizer.git`
2. **Install Dependencies:** `pip install -r requirements.txt`
3. **Setup Data:** `python data_setup.py`
4. **Run App:** `streamlit run app.py`

---

## 📂 Folder Architecture
- `data/`: Raw and processed datasets.
- `models/`: Serialized ML models (.pkl).
- `notebooks/`: Exploratory Data Analysis (EDA).
- `assets/`: Project screenshots and demo video.
