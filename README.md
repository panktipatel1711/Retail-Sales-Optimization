## 🚀 Key Features
- **Predictive Intelligence:** 7-Day multi-step forecasting.
- **Supply Chain Logic:** Automatic Safety Stock and Reorder Point (ROP) calculation.
- **Business UI:** Professional dark-themed dashboard for high data readability.
- **Modular Code:** Separated logic for Data Processing, ML, and UI.

---

## 🛠️ Technical Setup
1. **Prerequisites:** Python 3.9+
2. **Install Libraries:** `pip install -r requirements.txt`
3. **Data Generation:** `python data_setup.py`
4. **Launch App:** `streamlit run app.py`

---

## 📂 Folder Architecture
- `data/`: Raw sales datasets.
- `models/`: Serialized ML models (.pkl).
- `src/`: Core engine for inventory math.
- `assets/`: Demo video and screenshots.
